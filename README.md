# アプリケーション名
TabemonoReserch

# 概要
本アプリケーションは、PHPフレームワークであるLaravelおよびぐるなびのAPIを利用して、検索キーワードに応じて飲食店結果を返すLINEBotである。<br>
以下の項目について基礎的知識を得るために開発した。<br>
- Laravel
- Docker
- Laradock
- Heroku
- LINE Messaging API
- ぐるなび API

# システム要件
- MacOS
- Homebrew
- Docker
- Laradock v9.6
- Laravel v6.8
- LINEアカウント
- ぐるなびAPIアカウント
